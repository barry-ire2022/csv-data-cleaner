# CSV Data Cleaning Tool

This is a **Streamlit app** that helps clean CSV files quickly, without installing complex software.  

## Features

- Normalize dates in column `Date`
- Clean and validate emails in column `Email`
- Apply sentence case to comments in column `Comment`
- Trim whitespace in all cells
- Remove duplicate rows
- Fill all missing or empty cells with `N/A`
- Download cleaned CSV
